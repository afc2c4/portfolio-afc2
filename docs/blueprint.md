# **App Name**: ArtFolio

## Core Features:

- Editable Profile: Allow users to edit profile attributes such as name, bio, skills, and contact information.
- Post Management: Enable users to create, edit, and delete posts showcasing their work.
- Category Assignment: Allow users to categorize posts using predefined tags to enable search and filtering. The LLM is used as a tool to pre-populate the tags based on an image.
- Image Upload: Provide a feature to upload images for portfolio posts.
- Refactoring Support: Automated refactoring tools to assist with code improvements (using IDE plugins).
- Pattern Implementation: Guidance and suggestions for implementing design patterns within the codebase.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), evokes creativity.
- Background color: Very light grey (#F5F5F5), a muted tone with the same hue as the primary color.
- Accent color: Dark indigo (#3F51B5), contrasting brightness/saturation from primary.
- Font pairing: 'Belleza' for headlines (sans-serif) and 'Alegreya' for body (serif), adding style and readability.
- Simple, modern icons for profile and post management functions.
- Clean, grid-based layout for portfolio posts to showcase work effectively.
- Subtle transitions and hover effects to add interactivity and visual interest.
